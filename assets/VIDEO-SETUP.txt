TROUBLESHOOTING:
----------------
- If videos don't play: Check file names match exactly
- If videos are too large: Compress using HandBrake or similar
- If videos don't loop: Make sure they're in MP4 format
- Browser compatibility: Works best in Chrome, Firefox, Edge
