HOW TO TAKE SCREENSHOTS:
-------------------------

METHOD 1: Manual Screenshot
1. Open each website in browser
2. Press Win+Shift+S (Windows) or Cmd+Shift+4 (Mac)
3. Select full page
4. Save screenshot

METHOD 2: Online Tool (Easiest)
1. Go to: https://www.screenshotmachine.com/
2. Paste website URL
3. Click "Capture Screenshot"
4. Download image

METHOD 3: Browser Extension
1. Install "Full Page Screen Capture" extension
2. Open website
3. Click extension icon
4. Download screenshot
